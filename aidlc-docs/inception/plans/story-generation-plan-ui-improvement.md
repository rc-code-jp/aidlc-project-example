## Breakdown Options
- **User Journey-Based**: アップロード前、アップロード中、結果確認の体験で分ける。今回の UI 改善と相性が良い
- **Feature-Based**: アップロード、結果表示、状態表示の機能単位で分ける。画面部品の整理に向く
- **Persona-Based**: ユーザー種別が複数ある場合に有効。今回は優先度が低い
