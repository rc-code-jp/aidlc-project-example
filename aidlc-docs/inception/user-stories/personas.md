# Personas

## Persona 1: 一般利用者

### 概要
- Web ブラウザから画像をアップロードして、犬が写っているかを手早く確認したい利用者

### 特徴
- 技術的な設定は最小限で済ませたい
- 画像を送るだけで直感的に結果を見たい
- 長い説明より、数値と短いラベルで理解したい

### 目的
- 手元の画像に犬が写っているか、どの程度犬らしいかをすぐ知る

### 行動
- 単一画面で画像を選ぶ
- 判定完了まで待つ
- パーセントとラベルを見て結果を判断する

### 期待
- 操作が簡単であること
- 判定中かどうかが分かること
- エラー時にやり直し方法が分かること
