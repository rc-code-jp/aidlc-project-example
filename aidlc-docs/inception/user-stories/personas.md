# Personas

## Persona 1: 一般利用者

### 概要
- Web ブラウザから画像をアップロードし、見やすく気持ちの良い画面で犬判定結果を確認したい利用者

### 特徴
- 技術的な設定は最小限で済ませたい
- 画像を送る前に内容を見て安心したい
- 長い説明より、視覚的に分かりやすい結果表示を好む

### 目的
- 手元の画像に犬が写っているかを、迷わない操作と印象の良い画面で素早く知る

### 行動
- 単一画面で画像を選ぶ、またはドラッグアンドドロップする
- 選択画像のプレビューを確認する
- 判定完了まで待つ
- 強調表示されたパーセントとラベルを見て結果を判断する

### 期待
- 操作が簡単で、アップロード前の不安が少ないこと
- 判定中かどうかが視覚的に分かること
- 結果がひと目で理解できること
- エラー時にやり直し方法がすぐ分かること
