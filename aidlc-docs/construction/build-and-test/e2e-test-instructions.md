# E2E Test Instructions

## Purpose
利用者の主要フローを手動で確認する。

## Main Scenario
1. トップページを開く
2. 画像をドラッグアンドドロップ、またはクリックして選択する
3. プレビューが表示されることを確認する
4. 判定実行を押す
5. ローディング表示を確認する
6. パーセント、ラベル、理由が結果カード内に表示されることを確認する

## Error Scenario
1. API キー未設定または無効状態で起動する
2. 画像を送信する
3. エラー文言と再試行可能な UI を確認する

## Accessibility Scenario
1. キーボードだけでファイル入力とボタン操作が可能か確認する
2. 結果やエラーがテキストで読めることを確認する
3. ドラッグアンドドロップに頼らなくても同等操作が可能であることを確認する
